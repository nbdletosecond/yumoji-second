# yumoji

Input food emojis, output yummy recipes!

## Setup
```
python3 -m venv venv
pip install -r requirements-dev.txt -r requirements.txt
```

## Django-Tailwind Setup
```
python3 manage.py tailwind install
python3 manage.py tailwind start
```
